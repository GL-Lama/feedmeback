# Lama Feedmeback

## How to launch the project ?

+ install java 1.8 (setup JAVA_HOME for windows)
+ install maven (setup PATH for windows)

### Compilation

```sh
$ mvn compile
```

### Build

```sh
$ mvn package
```

### Launch the application

```sh
$ java -jar target/feedmeback-1.0-jar-with-dependencies.jar
```
